数学記号 Unicode 変換ツール

簡単な入力記法を、コピー可能なUnicode数学表現へ変換します。

入力すると自動変換されます。指数全体に + や − を含める場合は、x^{2n+1}のように波括弧で囲みます。分数指数はx^{fra 1,2}、シグマはsig k=1,n a[k]と入力できます。三角関数・対数関数の引数にはcos{fra nPi,6}のように分数を入れられます。コピーできない場合は、結果欄が選択されるので長押ししてコピーしてください。

入力例

int 1,3 x^2+x
→ ∫₁³(x²+x) dx
dif x^2+x
→ (d/dx)(x²+x)
nth,3 x+1
→ ₃√(x+1)
fra x+1,y-2
→ (x+1)/(y−2)
lim n,inf fra 1,n^2
→ lim(n→∞) 1/n²
a[n+1] <= b[n]
→ aₙ₊₁ ≤ bₙ
x^2n+1
→ x²ⁿ+1
x^{2n+1}
→ x²ⁿ⁺¹
x^{fra 1,2}
→ x¹ᐟ²
sig k=1,n a[k]
→ Σ(k=1,n)aₖ
fra 1,2^n+cos{fra nPi,6}
→ (1/2ⁿ)+cos(nπ/6)
theta^beta+alpha+Pi
→ θᵝ+α+π
